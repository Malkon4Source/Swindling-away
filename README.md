# Swindling-away

### Description
Just a simple shell script that horizontally centers ascii art saying "swindling away", as a fan-project for Swindle by KantianKant.

This was originally inspired by a reddit post in r/unixporn that contained the figlet output "Musling it". I thought it was a program drawing it on the terminal until I looked closer after I had made this and saw it was just figlet.

### Installation
Simply download the "swindling-away" file and run it. Make sure it's executable by running "chmod +x swindling-away". You can also move it to /bin if you wanna use it across your system, no matter directory.

Link to the actual Wayland Compositor repository: https://github.com/kantiankant/swindle
